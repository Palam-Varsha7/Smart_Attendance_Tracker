# Smart Attendance Tracker

## A Streamlit-based Attendance Management System

## Overview
Smart Attendance Tracker is a desktop application built using Streamlit that allows teachers to manage student attendance efficiently. It supports multiple subjects, periods, and provides attendance percentage calculation for each student.

## Features
- Add and manage student records
- Mark attendance by subject and period
- View complete attendance records
- Calculate attendance percentage per student
- Modify existing attendance records
- Delete incorrect records
- Data stored locally in CSV files

## Tech Stack
- Python
- Streamlit (UI Framework)
- Pandas (Data Management)
- CSV (Local Data Storage)
  
## How to Run
Install the required libraries using:

pip install streamlit pandas

Then run the app using:

streamlit run app.py

## Pages
- Add Student — Register new students
- Mark Attendance — Mark present or absent by subject and period
- View Records — See all records and attendance percentage
- Modify Record — Edit any existing entry
- Delete Record — Remove incorrect entries
